Century City Mall App

Project information

  • Category: iOS Mobile App
  • Company: MobileGroup Inc.
  • Client: Century City Mall (Century Propoerties)
  • Project date: 2014
  • Project URL: https://apps.apple.com/us/app/century-city-mall/id934284519?ls=1

• Developed an iOS mobile mall application which will be used by Century City Mall-goers as personal digital guide to the services of Century City Mall for the client, Century Group Properties, Inc. With the use of the app, the user has access to shops, restaurants, events, promos and the mall concierge, whether the user is online or offline. The user can also check the latest cinema schedules (and trailers). The user can also use the manual parking guide to remind him where he parked his car.
• Technology Used: Objective – C (iOS), web services (JSON, XML, SOAP), and external Youtube player API.
